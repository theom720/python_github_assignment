# Study Hours Tracker

## Description

This program asks the user how many hours they studied today, converts the input into a number, handles invalid input, and then calculates their projected weekly study hours. It helps users stay on track with their study goals.

## How to Run

Run the program using one of the following commands:

python app.py  
or  
python3 app.py

## Example Output

Enter hours studied today: 2  
You are on track to study 14 hours this week.
